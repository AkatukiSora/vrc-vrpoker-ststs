# 5. 機会ベースメトリクスと信頼度閾値

Date: 2026-02-27
日付: 2026-02-27

## Status

accepted

## 背景

プレイヤーの傾向を正しく把握するには、単純なカウントではなく機会数を分母にした比率が必要。
また、サンプル数が少ない場合は過信を避ける表示が求められる。

## 決定

統計メトリクスは機会ベース（`count / opp`）で算出し、
信頼度の閾値は hand-frequency で `n >= 200`、situational で `n >= 50` とする。
閾値未満の値は「参考値」として表示する。

## 結果

### メリット
- 意味のある比較ができ、誤解を減らせる。
- UI における指標の信頼度を一貫して伝えられる。

### トレードオフ
- サンプル数が少ないユーザーへの指標表示が控えめになる。
- 閾値の見直しには統計と UI 双方の整合性が必要。
